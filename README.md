# MEGA WIRELESS — MASTER VISUAL LOCK V1

This package locks the public website to the approved master direction.

## Locked design rules
- Owner/technician scene appears only in the hero.
- Phone cards never reuse the owner's photo.
- Speaker images use contained or approved promo crops.
- Carrier artwork is not stretched into banners.
- Admin and POS are intentionally not exposed in the public navigation.
- No customer data is stored in this package.

## Current scope
This is the approved public front-end baseline only. It is safe for public deployment because it contains no customer records, authentication tokens, or private data.

## Next production phase
- Secure database
- Employee authentication
- Admin CMS
- POS and inventory
- Customer and repair-ticket data with server-side authorization
- Audit logs and backups

Do not enter customer data into static HTML or browser localStorage.
